## 🎯 5가지 섹션 구조

모든 지표는 다음 5개 섹션으로 구성됩니다:

### 1. **core_definition** (핵심 정의)
- **목적**: 지표가 무엇인지 명확히 정의
- **내용**:
  - 지표의 정의 (1-2문장)
  - 측정 방법 (조사 대상, 조사 항목 등)
  - 표현 방식 (지수, 비율, 절대값 등)
- **길이**: 3-5줄
- **예시**:
  ```
  ISM 제조업 PMI는 미국 제조업체 구매관리자들을 대상으로 한
  설문조사 기반 경기 선행지표입니다. 신규 주문, 생산, 고용, 재고, 배송 속도 등
  5개 항목을 종합하여 0~100 사이의 지수로 표현합니다.
  ```

### 2. **interpretation_guide** (해석법)
- **목적**: 숫자를 어떻게 읽는지 가이드
- **내용**:
  - 구간별 해석 (높음/중립/낮음 기준)
  - 변화의 의미 (상승/하락 시 의미)
  - 임계점 (경계 수치)
- **형식**: 불릿 포인트 (`•`) 사용
- **길이**: 5-7줄
- **예시**:
  ```
  • 50 이상: 제조업 경기 확장 (전월 대비 개선)
  • 50: 중립 (변화 없음)
  • 50 미만: 제조업 경기 위축 (전월 대비 악화)
  • 55 이상: 강한 확장세 (경기 과열 가능성)
  • 45 미만: 심각한 위축 (경기침체 우려)
  ```

### 3. **context_meaning** (의미·맥락)
- **목적**: 경제적 맥락과 중요성 설명
- **내용**:
  - 경제에서의 비중/역할
  - 연준 통화정책과의 관계
  - 선행/동행/후행 지표 특성
  - 주의사항 (변동성, 수정치, 한계점 등)
- **길이**: 3-4줄
- **예시**:
  ```
  제조업은 미국 GDP의 약 11%를 차지하지만, 고용과 설비투자에 미치는
  파급효과가 커서 경기 판단에 매우 중요한 지표입니다. 특히 연준의 통화정책
  결정 시 핵심 참고 지표로 활용됩니다. PMI는 실제 생산 데이터보다 1-2개월
  앞서 움직이는 선행지표 특성이 있습니다.
  ```

### 4. **market_reaction** (시장 반응·투자 적용)
- **목적**: 실전 투자 전략 제시
- **내용**:
  - 예상보다 높을 때 시장 반응
  - 예상보다 낮을 때 시장 반응
  - 자산별 영향 (주식, 채권, 달러, 금 등)
  - 구체적 투자 전략
- **형식**: 불릿 포인트 사용
- **길이**: 4-6줄
- **예시**:
  ```
  • 예상보다 높을 때: 주식 상승(특히 산업재, 소재), 달러 강세, 금리 상승 압력
  • 예상보다 낮을 때: 주식 하락(경기 둔화 우려), 채권 선호, 달러 약세
  • 50 근처 등락: 경기 전환점 신호 → 변동성 확대 가능
  • 투자 전략: 55 이상 시 경기민감주 매수, 45 미만 시 방어주 선호
  ```

### 5. **additional_info** (확인 정보)
- **목적**: 발표 시기 및 참고 정보
- **내용**:
  - 발표 시기 (매월/분기, 요일, 시간)
  - 조사 기간
  - 출처 기관
  - 역사적 배경 (선택)
  - 관련 지표 (보완/대체 지표)
- **형식**: 불릿 포인트 사용
- **길이**: 4-5줄
- **예시**:
  ```
  • 발표 시기: 매월 첫 영업일 오전 10시 (ET)
  • 조사 기간: 전월 말 기준
  • 출처: Institute for Supply Management (ISM)
  • 역사: 1948년부터 발표 시작, 70년 이상의 신뢰도
  • 관련 지표: ISM 비제조업 PMI, S&P Global Manufacturing PMI
  ```

## 📋 작성 템플릿 (복사해서 사용)

```python
'indicator-id': {
    'name': 'English Name',
    'name_ko': '한글 지표명',
    'category': 'business',  # business/employment/interest/trade/inflation/policy
    'reverse_color': False,  # True면 낮을수록 좋음 (실업률 등)

    'interpretation': {
        'core_definition': """
            [지표가 무엇인지 3-5줄로 정의]
            [측정 방법, 조사 대상, 표현 방식 포함]
        """,

        'interpretation_guide': """
            • [구간 1]: [해석]
            • [구간 2]: [해석]
            • [구간 3]: [해석]
            • [특이점]: [해석]
        """,

        'context_meaning': """
            [경제적 맥락 3-4줄]
            [GDP 비중, 연준 정책 관련성, 선행/동행/후행 특성]
            [주의사항: 변동성, 수정치, 한계점 등]
        """,

        'market_reaction': """
            • 예상보다 높을 때: [주식/채권/달러/금 반응]
            • 예상보다 낮을 때: [주식/채권/달러/금 반응]
            • [임계점 돌파 시]: [특이 반응]
            • 투자 전략: [구체적 전략 제시]
        """,

        'additional_info': """
            • 발표 시기: [매월/분기, 요일, 시간대]
            • 조사 기간: [조사 대상 기간]
            • 출처: [발표 기관명]
            • [특이사항]: [역사, 수정치 여부 등]
            • 관련 지표: [보완 지표 나열]
        """
    }
},
```

### 지표 정보 조회 사이트
1. **Investing.com**: https://www.investing.com/economic-calendar/
   - 발표 일정, 예측값, 실제값, 히스토리 차트
2. **TradingEconomics**: https://tradingeconomics.com/
   - 지표 정의, 역사적 데이터, 국가별 비교
3. **FRED (St. Louis Fed)**: https://fred.stlouisfed.org/
   - 공식 데이터, 장기 차트, 출처 확인
4. **Investopedia**: https://www.investopedia.com/
   - 초보자 친화적 설명, 투자 시사점
5. **U.S. Bureau of Labor Statistics**: https://www.bls.gov/
   - 고용지표 공식 출처 (실업률, 비농업고용 등)
6. **U.S. Bureau of Economic Analysis**: https://www.bea.gov/
   - GDP, PCE 등 공식 출처

- **핵심 정의**: Investopedia + TradingEconomics 참고

- **맥락**: FRED 데이터 + 연준 발언록
- **시장 반응**: Bloomberg, CNBC 뉴스 패턴 분석

- **핵심 정의**: 조사한 내용을 3-5줄로 요약

- **확인 정보**: 발표 시기, 출처, 관련 지표 나열

### 4단계: 검증 (5분)

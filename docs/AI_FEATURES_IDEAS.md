# Investment App AI Features Ideas

> 경제지표 대시보드 + 투자 분석을 고도화하기 위한 AI 기능 아이데이션

**작성일**: 2026-02-02

---

## 1. 방향성

Investment App의 AI는 **지표 해석 + 투자 가설 검증 + 리스크 감지**를 중심으로 설계한다.

핵심 목표:
- 지표 데이터를 빠르게 이해
- 투자 가설의 논리적 완성도 향상
- 시나리오/리스크 요약 자동화

---

## 2. 우선순위 기능 아이디어

### A. 지표 해석/요약
- **발표 요약**: 실제치/예측치/서프라이즈 기반 한 줄 해석
- **변화 포인트 강조**: 전월 대비/추세 요약
- **지표 캘린더 브리핑**: 다음 발표 일정 + 예상 영향

### B. 투자 가설 작성 지원 (Analysis 탭 강화)
- **가설/리스크 초안 생성**
- **논리 일관성 체크**: 필드 간 모순/누락 감지
- **빈 필드 추천 채우기**

### C. 시나리오 분석 지원
- **최선/기본/최악 시나리오 내러티브 자동 생성**
- **목표가/손절가 근거 요약**

### D. 리스크/경고 시스템
- **지표 급변 경고**
- **분석 탭 리스크 필드 누락/모순 감지**

### E. 리서치 요약/지식화
- **뉴스/리서치 요약**
- **핵심 인사이트 카드화**

---

## 3. 도입 순서 제안 (Phase 기반)

### Phase 1 — 지표 요약 + 가설 작성 지원
- 발표 요약
- 가설/리스크 초안 생성

### Phase 2 — 시나리오/리스크 자동화
- 시나리오 내러티브
- 리스크 누락/모순 감지

### Phase 3 — 리서치 지식화
- 뉴스 요약
- 인사이트 카드

- UI 적용 위치 정의 (Analysis 탭/지표 카드/리포트)

## 2. 새로운 접근: 흐름(Flow) 시스템

### 2.1 핵심 철학
- **점수화 대신 방향성**: ↑ ↓ → 로 직관적 표시
- **3가지 질문**: 방향? 속도? 누가/무엇이?
- **Less is More**: 47개 → 12-15개 핵심 지표

### 2.2 지표종류별 핵심 지표

#### 경기지표 (경제 확장/수축) — 8개 → 4개 선별

**선별 기준**: 흐름 질문(확장/수축? 속도? 주체?)에 직접 답할 수 있는가?

| 지표 | 역할 | 흐름 질문 | 판정 | 이유 |
|------|------|-----------|------|------|
| **ISM 제조업 PMI** | 선행 | 확장/수축, 속도 | ✅ 필수 | 50 기준 명확, 시장 민감도 최고 |
| **ISM 서비스업 PMI** | 선행 | 확장/수축, 주체 | ✅ 필수 | 미국 경제 70%, 제조업보다 중요 |
| **소매판매 MoM** | 동행 | 소비 흐름 | ✅ 필수 | GDP 70% = 소비, 핵심 엔진 |
| **미시간 소비자심리** | 선행 | 심리 + 기대 인플레 | ✅ 필수 | Fed 주목, 인플레 기대 포함 |
| GDP QoQ | 후행 | 확인용 | △ 선택 | 분기별, 이미 반영된 후 |
| 산업생산 | 동행 | 제조업 체력 | ❌ 제외 | PMI가 이미 커버, 중복 |
| CB 소비자 신뢰 | 선행 | 노동 체감 | ❌ 제외 | 미시간과 중복 (하나만) |
| OECD CLI | 선행 | 글로벌 비교 | ❌ 제외 | 단기 투자에 덜 유용 |
| LEI | 선행 | 종합 추세 | ❌ 제외 | 이미 발표된 지표 합성, 시장 충격 낮음 |

**최종 경기지표 (4+1개)**:
- **필수 4개**: ISM 제조업 PMI, ISM 서비스업 PMI, 소매판매, 미시간 소비자심리
- **선택 1개**: GDP (분기별 확인용)

**흐름 해석 예시**:
```
경기: 수축 진입 ▼ (제조업 PMI 48.5)
속도: 둔화 중 → (전월 대비 -1.2)
주체: 서비스업 버팀 ▲ (52.1) / 제조업 약화 ▼
소비: 강세 유지 ▲ (소매판매 +0.4%)
심리: 낙관 → 비관 전환 ▼ (미시간 67.8)
```

#### 고용지표 (돈 벌 기회)
| 핵심 질문 | 핵심 지표 | 읽는 법 |
|-----------|-----------|---------|
| **일자리 늘어나는가?** | Nonfarm Payrolls | +면 증가, -면 감소 |
| **실업률은?** | Unemployment Rate | 낮을수록 좋음 |
| **임금은?** | Avg Hourly Earnings YoY | 임금 상승률 |
| **해고 늘어나는가?** | Initial Jobless Claims | 높을수록 나쁨 |

**흐름 해석**: 일자리 증가 중? 실업률 상승/하락? 임금 오르는 속도? 해고 물결?

#### 금리지표 (돈의 가격)
| 핵심 질문 | 핵심 지표 | 읽는 법 |
|-----------|-----------|---------|
| **기준금리 방향?** | Federal Funds Rate | 인상/동결/인하 |
| **시장금리는?** | 10Y Treasury Yield | 장기 금리 방향 |
| **경기침체 신호?** | Yield Curve (10Y-2Y) | 음수면 역전 (위험) |

**흐름 해석**: Fed가 돈을 푸는가 조이는가? 시장은 침체를 예상하는가?

#### 물가지표 (돈의 가치)
| 핵심 질문 | 핵심 지표 | 읽는 법 |
|-----------|-----------|---------|
| **인플레이션 수준?** | Core CPI YoY | 2% 목표 대비 |
| **인플레이션 방향?** | CPI MoM | +면 상승, -면 하락 |

**흐름 해석**: 물가 오르는 중? 속도가 빨라지나 느려지나?

#### 심리지표 (시장 분위기)
| 핵심 질문 | 핵심 지표 | 읽는 법 |
|-----------|-----------|---------|
| **공포/탐욕?** | VIX | >30 공포, <15 탐욕 |
| **소비자 마음?** | Michigan Sentiment | 높을수록 낙관 |

**흐름 해석**: 사람들이 무서워하는가 탐욕스러운가? 소비할 마음이 있는가?

## 3. 사이클 시스템 vs 흐름 접근법 비교

| 관점 | 사이클 시스템 (현재) | 흐름 접근법 (제안) |
|------|----------------------|-------------------|
| **복잡도** | 높음 (점수화, 가중치) | 낮음 (방향/속도/주체) |
| **정량적** | 52.3점, 국면 판별 | ↑확장 중, 속도 둔화 |
| **투자 연결** | 점수 구간별 추천 | 흐름 조합으로 판단 |
| **유지보수** | 어려움 (로직 복잡) | 쉬움 (해석 로직) |
| **직관성** | 낮음 | 높음 |
| **신뢰도** | 백테스트 필요 | 상식적 판단 |

## 9. 현재 구조 및 전환 옵션

### 9.1 현재 구조

```
CyclePanel (사이클 점수)
  ↓
IndicatorGrid (카드 + 자세히)
  ↓
IndicatorChartPanel (수치/차트/해석 탭)
```

### 9.2 전환 옵션

#### Option A: 폐기 + 흐름 섹션으로 대체
- `CyclePanel`, 기존 상세 구조를 대폭 축소하고 `FlowDashboard` 중심으로 재구성
- 장점: 가장 단순함, 구조 정리 효과 큼
- 단점: 기존 구현 자산(카드/차트/해석) 재사용 폭이 작음

#### Option B: 유지 + 흐름 섹션 추가 (상단)
- 상단에 `FlowDashboard` 추가
- 기존 `IndicatorGrid`, `IndicatorChartPanel`은 유지
- 지표는 47개에서 핵심 15개 중심으로 필터
- 장점: 기존 투자 보존, 빠른 전환
- 단점: 요약/상세 중복 가능성

#### Option C: 병합 (흐름 = 상단, 상세 = 펼치기)
- 상단 요약은 `FlowDashboard`
- 상세는 아코디언/펼치기 구조로 접근
- 장점: 단순함과 상세 접근성 균형
- 단점: UI 상태 관리/아코디언 구현 필요

### 9.3 추천안

**권장: Option B (유지 + 추가)**

이유:
1. 기존 컴포넌트(`EnhancedIndicatorCard`, `IndicatorChartPanel`) 재사용 가능
2. 작업량이 적고 전환 리스크가 낮음
3. 사용자 반응 확인 후 Option C 또는 A로 단계 전환 가능

즉시 반영 범위:
- `CyclePanel` 역할을 `FlowDashboard`로 보강 또는 대체
- `IndicatorGrid` 기본 노출을 핵심 15개(우선순위 4+1 포함)로 축소
- 상세 차트 패널은 유지

## 11. 지표 총정리 (카테고리별 확정안)

### 11.1 경기 (Growth)
- **필수 4**: ISM 제조업 PMI, ISM 서비스업 PMI, 소매판매 MoM, 미시간 소비자심리
- **선택 1**: GDP QoQ (분기 확인용)
- **제외 후보**: 산업생산, CB 소비자신뢰, OECD CLI, LEI

### 11.2 고용 (Labor)
- **필수 4**: Initial Jobless Claims(4주평균), NFP(3개월평균), 실업률(U3+Sahm Rule), 평균시간당임금(AHE)
- **선택 1**: LFPR(가능하면 prime-age 25-54)
- **장기 분리**: 인구증가율 (단기 대시보드에서 분리)

### 11.3 금리 (Rates)
- **필수 5**: FFR, 2Y Treasury, 10Y Treasury, 10Y-2Y Spread, 10Y Real Yield(TIPS/DFII10)
- **운영 포인트**: FFR은 FOMC 이벤트 중심, 나머지는 일간/실시간 추세 중심

### 11.4 물가 (Inflation)
- **필수 5**: Core CPI MoM, Core PCE YoY, 5Y Breakeven(T5YIE), Brent, BCOM/CRB
- **선택 2**: Core PPI MoM, 미시간 5Y 기대인플레
- **비고**: Headline CPI는 착시 가능성이 있어 보조 해석용

### 11.5 무역/달러/공급망 (Trade)
- **필수 6**: DXY, USD/KRW, 미국 상품무역수지, Inventory-to-Sales Ratio, GSCPI, FBX(또는 BDI 1개 선택)
- **선택**: 경상수지(분기), REER(월간), XPI/TOT(월간 구조 해석용)
- **비고**: XPI/TOT는 단기 트레이딩보다 중기 경쟁력/구매력 분석에 중요

### 11.6 신용 (Credit)
- **필수 4**: HY OAS, IG OAS, BBB-10Y Spread, 커브 지표(10Y-2Y 등 핵심 1개)
- **선택 2**: SLOOS, 연체율/충당금(분기)
- **제외 후보**: 등급별 스프레드 과다 중복(AA~CCC 전부 상시 노출)

### 11.7 심리 (Sentiment)
- **필수 4**: VIX, VIX Term Structure(VIX3M-VIX), Put/Call Ratio(5일 평균), AAII Bull-Bear Spread
- **선택 2**: NAAIM Exposure, Fear & Greed
- **제외 후보**: 유사 설문 다중 중복, 해석 기준 불명확한 감성 지표

### 11.8 운영 원칙
1. 카테고리별 `필수`는 기본 화면 고정 노출
2. `선택`은 Advanced(접힘)로 배치
3. 장기 구조 지표(인구, REER, 경상수지 등)는 별도 섹션 분리
4. 단계적 전환은 Option B(유지+추가) 기준으로 진행
